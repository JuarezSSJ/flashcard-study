O que é uma Tautologia?

Uma proposição composta que é sempre verdadeira, independentemente dos valores lógicos de suas proposições simples. Exemplo clássico: p ∨ ~p.

O que é uma Contradição?

Uma proposição composta que é sempre falsa, independentemente dos valores lógicos de suas proposições simples. Exemplo clássico: p ∧ ~p.

Ordem de precedência dos conectivos

1º Negação (¬)
2º Conjunção (∧)
3º Disjunção (∨)
4º Condicional (→)
5º Bicondicional (↔)
